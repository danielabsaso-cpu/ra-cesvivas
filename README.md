# Vivero Sustentable "Raíces Vivas"

Bienvenido al repositorio del sitio web del Vivero Sustentable "Raíces Vivas".

## Descripción

Este proyecto es un sitio web para el vivero "Raíces Vivas", donde los usuarios pueden explorar catálogos de plantas, conocer sobre nosotros, ver galerías y contactarnos.

## Estructura del Proyecto

- **HTML:** Archivos para cada página (Inicio, Sobre Nosotros, Catálogo, Galería, Contacto, Ingresar)
- **CSS/SCSS:** Estilos organizados en múltiples archivos SCSS y compilados en CSS
- **Imágenes:** Logos, plantas y galería
- **Y más**
